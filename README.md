# commute-calculator

A simple static commute calculator built for GitHub Pages.

## Usage

- Open `index.html` in a browser.
- Use the default twice-per-week commute schedule with `3:40` hours per day.
- Adjust days per week, commute duration, and weeks per month to calculate totals.

## GitHub Pages

1. Push this repository to GitHub.
2. In the repository settings, enable GitHub Pages from the `main` branch.
3. The site will be served from:
   `https://<your-github-username>.github.io/commute-calculator/`

The calculator is ready as a static page and works without any additional build steps.
